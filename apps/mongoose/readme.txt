How to build and run mongoose server (with aget).

1. Buile server and client.
$ cd $XTERN_ROOT/apps/aget
$ ./mk
$ cd $XTERN_ROOT/apps/mongoose
$ ./mk

2. Download the input file.
$ cd $XTERN_ROOT/apps/mongoose
$ wget http://llvm.org/releases/2.7/llvm-gcc-4.2-2.7.source.tgz

3. Run
$ cd $XTERN_ROOT/apps/mongoose
$ ./run
